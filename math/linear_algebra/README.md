## linear Algebra

## Description
This project covers fundamental linear algebra concepts implemented in Python.
It explores vectors, matrices, and numerical operations using NumPy.


## Requirements
- Ubuntu 16.04 LTS
- Python 3.5
- NumPy 1.15
- pycodestyle 2.5

## Files
| File | Description |
|------|-------------|
| `0-slice_me_up.py` | Slicing a Python list in different ways |
| `1-trim_me_down.py` | Extracting middle columns of a matrix |
| `2-size_me_please.py` | Calculating the shape of a matrix |
| `3-flip_me_over.py` | Transposing a 2D matrix |
| `4-line_up.py` | Adding two arrays element-wise |
| `5-across_the_planes.py` | Adding two 2D matrices element-wise |
| `6-howdy_partner.py` | Concatenating two arrays |
| `7-gettin_cozy.py` | Concatenating two 2D matrices along an axis |

| `9-let_the_butcher_slice_it.py` | Slicing numpy arrays |
| `10-ill_use_my_scale.py` | Shape of a numpy.ndarray |

| `13-cats_got_your_tongue.py` | Concatenating numpy arrays |
